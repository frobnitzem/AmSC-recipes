To use these examples, you will need to navigate to <https://api.i2-core.american-science-cloud.org> and click the "Key Manager" banner to get an API key.
Then save this value in the `AMSCAI_API_KEY` environment variable,
 
    AMSCAI_API_KEY=sk-<your key value>

Note that the ALCF LLM inference endpoints can also be used by those with
DOE lab credentials, after gaining authorization from ANL.
API access to those endpoints is well-documented [here](https://docs.alcf.anl.gov/services/inference-endpoints/#api-access).
