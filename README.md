# AmSC Recipes

This is a collection of "Hello World" recipes for using IRI and AmSC tools.

The development status of each tool is listed below.

 - iri-facility-api/hello-status (uses status.py): working

 - inference/completions.py: working

 - inference/completions.sh: working

For examples marked working, please open a github issue if something
does not work as expected.

## Installation

Each subdirectory contains `requirements.txt` listing python modules
required in order to run the examples.
